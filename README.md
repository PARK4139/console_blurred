### 프로젝트 기본설명
 ```bash
"토이 프로젝트 진행"
-주제
    윈도우 기반 자동화 프로그램
-개발환경
    FE : Vercel/Node.js/React/Next.js
    BE : AWS EC2/Ubuntu/Nginx/Docker/Uvicorn/Fastapi
-참여인원 및 리소스
    1인 개발
-개발기간
    2022.12.xx - 2024.02.08 
-결과
    https://github.com/PARK4139/archive_py
-성과
    컴퓨터 반복업무 효율개선

    파일분류 자동화
    파일백업 자동화
    유튜브 영상클립 다운로드
```

